## ⬇️ PROMPT (여기부터 복사)

한국의 방산·항공우주 시험계측 전문기업 **(주)넥시스(NEXYS — New Experiment System)**의 기업 홈페이지를 디자인해줘. 결과물은 정적 HTML + CSS(+JS, SVG 에셋)로, 이후 Next.js로 이식할 거야.

### 가장 중요한 지시 — 레퍼런스를 그대로 가져와라
첨부한 **ARX Robotics(arx-robotics.com, /mithra-os)**의 **컨셉·레이아웃·폰트 스타일·인터랙션을 거의 그대로 차용**해줘. 새로운 디자인을 발명하지 말고, ARX의 디자인 언어를 넥시스 콘텐츠에 입히는 작업이야. 단, **딱 하나만 바꾼다: ARX의 브랜드 포인트 컬러(그린/올리브 군용 톤)를 넥시스의 레드 #E60012로 전면 치환**. 그 외 비주얼 시스템(여백, 타이포 스케일, 그리드, 카드 형태, 모션, 스크롤 연출)은 ARX를 따라가.

### ARX에서 반드시 가져올 요소
- **타이포그래피**: 지오메트릭 산세리프(Inter / Manrope 계열), 초대형·볼드 헤드라인, 넉넉한 letter-spacing의 대문자 섹션 라벨, 본문은 절제된 그레이
- **컬러 운용**: 순수 블랙만 쓰는 게 아니라 ARX처럼 **다크 섹션 + 라이트(오프화이트) 섹션을 번갈아** 배치하는 리듬. 포인트 레드는 CTA·라벨·강조선·hover에만 희소하게
- **레이아웃**: 넓은 마진, 큰 풀블리드 이미지 카드, 좌우 비대칭 텍스트/이미지 블록
- **내러티브 구조**: ARX의 "The problem → Today's Challenges(01·02·03·04 번호 매긴 카드) → Our Solution → Industries(가로 슬라이더)" 흐름을 넥시스 버전으로 재현
- **인터랙션**: 부드러운 스크롤 reveal(페이드+슬라이드), 솔루션 카드 가로 슬라이더/탭, sticky 헤더(스크롤 시 배경 채워짐), 큰 hover 상태 전환, 메가 푸터
- **모션 톤**: 화려한 파티클보다 ARX처럼 **차분하고 고급스러운 전환**. 이미지 위주, 모션은 절제

### 넥시스 콘텐츠 매핑 (ARX 구조에 끼워넣기)
- **히어로**: ARX의 풀스크린 이미지 히어로처럼. 헤드라인 "Trust Your Idea & Technology", 서브 "Military & Aerospace Test Solution / HILS·SIL System / Test Automation & Engineering", CTA 2개
- **The problem → challenges**: 방산 시험의 난제(실장비 시험의 비용·위험, 통합 검증 복잡성, 국산화 필요성 등)를 01~04 번호 카드로
- **Our Solution**: 3사업부(시스템 / 엔지니어링 / 국방·항공기술 연구소)를 ARX의 솔루션 카드 형식으로
- **Industries / Products 슬라이더**: ARX의 가로 산업 슬라이더 자리에 → 넥시스 13개 구축 사례(VeriStand HILS, KF-X 연료리그, 200kg 수송드론 HILS, OPPAV Ironbird 등)를 가로 슬라이드 카드로
- **숫자 섹션**: 업력 9년+, 구축사례 13+, 파트너 15+, ISO 3종
- **파트너**: 국방·항공(ADD, 한화에어로, LIG넥스원, KAI, KARI, 현대로템 등) 로고 그리드/마퀴
- **푸터**: ARX식 메가 푸터(사이트맵 + 연락처 + 슬로건)

### 브랜드 규칙 (절대)
- 컬러: 레드 **#E60012** + 블랙 + 화이트/오프화이트만. 그린 계열 절대 금지
- 슬로건: "Trust Your Idea & Technology"
- 로고 모티프: 기울어진 NXS 레터마크(레드)

### 디자인할 페이지 (데스크톱 + 모바일)
1. 홈 (위 ARX 구조 전체)
2. 회사소개 (인사말 + 가치 3카드 + 2017→2024 타임라인 + 인증)
3. 구축 사례 목록 (필터 + 카드 그리드)
4. 사례 상세 1개 샘플 ("KF-X 연료리그시험장치 제어시스템")
5. 문의 (연락처 + 폼)

### 산출물 형식
- 페이지별 HTML + 공용 CSS + 공용 JS, 재사용 SVG 에셋
- 이미지 자리는 ARX처럼 큰 풀블리드 placeholder(다크 그라데이션 + 라벨)
- 모든 텍스트는 한국어 실제 카피 (Lorem ipsum 금지)
- ARX의 반응형 거동(모바일 햄버거 → 오버레이 메뉴 포함) 그대로
